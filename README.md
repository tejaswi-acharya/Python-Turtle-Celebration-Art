#  Python Turtle Celebration Art

This is a **Python Turtle graphics project** that draws colorful spirals, balloons, flowers, cakes, hearts, and a "Happy Birthday" message. It was a creative experiment in using loops, colors, and custom functions in Turtle to create animated graphics.

> Made in **Python** using the **Turtle module**. Originally created about a year ago in Jupyter Notebook.

---

## Features

- Multiple **spirals and patterns** drawn with loops and colors  
- **Balloon shapes** with layered colors  
- **Flower and leaf designs** using functions and loops  
- **Heart and cake illustrations** with layered colors  
- Custom **alphabet drawing functions** to write "Happy Birthday"  
- Fully interactive, the screen waits for a click to exit
